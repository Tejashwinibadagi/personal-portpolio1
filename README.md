ersonal Portfolio Website

Overview
This is my personal portfolio website, showcasing my projects, skills, and profile. It is designed using HTML, CSS, and JavaScript, and includes an interactive gallery and a section for personal information.
Features
Responsive layout for desktop and mobile devices
Profile section with personal introduction
Projects gallery with images and descriptions
Interactive "About Me" section with pastel-colored design (#E1DACA)
Smooth scrolling and clickable navigation

Folder Structure
portfolio/
│
├─ index.html       # Main HTML file
├─ style.css        # CSS styling
├─ script.js        # JavaScript for interactivity
├─ images/          # Folder containing all images (profile & project images)
│   ├─ profile.jpg
│   ├─ project1.jpg
│   └─ project2.jpg
└─ README.md        # Project overview

How to Use
Clone the repository to your local machine.
Make sure the images folder contains your profile picture and project images.
Open index.html in any browser to view the website.

Technologies Used
HTML5 – Structure of the website
CSS3 – Styling, including pastel colors and layout
JavaScript – Interactivity (clickable sections, gallery effects)
